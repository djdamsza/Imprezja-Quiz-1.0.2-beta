## Optymalizacja przed wstawieniem na stronę

Pliki są duże. Przed publikacją na WordPress:

1. **TinyPNG** (tinypng.com) lub **Squoosh** (squoosh.app) – kompresja PNG
2. Dla banera: zmniejsz szerokość do max 1200–1400 px (dla web)
3. Dla ikony: 512×512 px lub 256×256 px wystarczy

## Wstawianie w WordPress

- **Baner:** Media → Dodaj nowe → wgraj plik → wstaw na stronę jako obrazek nad sekcją „Pobierz”
- **Ikona:** Można użyć jako logo w nagłówku lub favicon (Appearance → Customize → Site Identity)
- **Promo:** Do postów na Facebooku, Instagramie, w materiałach promocyjnych
