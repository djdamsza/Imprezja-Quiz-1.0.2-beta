Pobierz Imprezja Quiz 1.0.4

Imprezja Quiz – quiz i głosowanie na imprezach. Działa offline w sieci lokalnej. 14 dni trial – pełna funkcjonalność.

Wymagania: Windows 10/11 (64-bit) lub macOS 10.13+

  • ⬇️ Windows – Imprezja Quiz Setup 1.0.4.exe
  • ⬇️ macOS Apple Silicon (M1/M2/M3) – Imprezja Quiz 1.0.4
  • ⬇️ macOS Intel – Imprezja Quiz 1.0.4

Instalacja: Program instaluje się na komputerze. Po instalacji znajdziesz go w menu Start (Windows) lub Launchpad / Aplikacje (macOS). Skrót na pulpicie (Windows) tworzony automatycznie.

Problem z odinstalowaniem na Windows? Pobierz napraw-uninstaller.bat i uruchom jako administrator.

Instalator Windows może być oznaczony przez antywirus – zobacz FAQ.
