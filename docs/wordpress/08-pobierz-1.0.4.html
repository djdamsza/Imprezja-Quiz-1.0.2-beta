Pobierz Imprezja Quiz 1.0.4

Imprezja Quiz – quiz i głosowanie na imprezach. Działa offline w sieci lokalnej. 14 dni trial – pełna funkcjonalność.

Wymagania: Windows 10/11 (64-bit) lub macOS 10.13+

  • ⬇️ Windows – Imprezja Quiz Setup 1.0.4.exe
  • ⬇️ macOS Apple Silicon (M1/M2/M3) – Imprezja Quiz 1.0.4
  • ⬇️ macOS Intel – Imprezja Quiz 1.0.4

Instalacja programu

Imprezja Quiz instaluje się na komputerze jak standardowa aplikacja:

  • Windows: Uruchom pobrany plik .exe, postępuj według instrukcji. Po instalacji program znajdziesz w menu Start oraz na pulpicie (skrót).
  • macOS: Otwórz pobrany plik .dmg, przeciągnij aplikację do folderu Aplikacje. Program znajdziesz w Launchpad oraz w Finderze → Aplikacje.

Program działa w sieci lokalnej – komputer z aplikacją pełni rolę serwera, gracze łączą się przez przeglądarkę na telefonach (bez instalacji).

Problem z odinstalowaniem na Windows?

Jeśli miałeś problem z odinstalowaniem starszej wersji (błąd NSIS, komunikat „zamknij aplikację”), pobierz i uruchom plik naprawczy:

⬇️ napraw-uninstaller.bat

Instrukcja: Pobierz plik, kliknij prawym przyciskiem myszy → Uruchom jako administrator. Poczekaj na zakończenie, potem uruchom instalator ponownie.

Instalator Windows może być oznaczony przez antywirus – zobacz FAQ.
