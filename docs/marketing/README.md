## Logo i badge (bez tła)

| Plik | Opis | Format |
|------|------|--------|
| `imprezja-quiz-logo.svg` | Logo „Imprezja Quiz” | SVG – **przezroczyste tło** |
| `imprezja-quiz-badge-14dni.svg` | Badge „14 DNI ZA DARMO” | SVG – **przezroczyste tło** |
| `imprezja-quiz-logo.png` | Logo (wersja rastrowa) | PNG – do usunięcia tła w [remove.bg](https://remove.bg) |
| `imprezja-quiz-badge-14dni.png` | Badge (wersja rastrowa) | PNG – do usunięcia tła w [remove.bg](https://remove.bg) |

**PNG bez tła:** Otwórz plik SVG w Figmie, Canva lub Inkscape i wyeksportuj jako PNG z przezroczystym tłem. Alternatywnie: wgraj PNG na [remove.bg](https://remove.bg) i pobierz wersję bez tła.
