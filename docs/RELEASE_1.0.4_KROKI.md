# Release 1.0.4 – kroki do publikacji

## 1. Utwórz Release na GitHub

1. Wejdź na: https://github.com/djdamsza/Imprezja-Quiz-1.0.2-beta/releases/new
2. **Tag:** `v1.0.4` (utwórz nowy)
3. **Tytuł:** `Imprezja Quiz 1.0.4`
4. **Opis:** np. „Naprawa instalatora Windows (komunikat o zamknięciu aplikacji)”

## 2. Wgraj pliki do Release

Przeciągnij do sekcji „Attach binaries”:

- `dist/Imprezja Quiz Setup 1.0.4.exe` (Windows)
- `dist/Imprezja Quiz-1.0.4-arm64.dmg` (macOS Apple Silicon)
- `dist/Imprezja Quiz-1.0.4.dmg` (macOS Intel)

**Uwaga:** GitHub może zmienić nazwy plików (spacje → kropki). Linki w HTML używają nazw z Release: `Imprezja.Quiz.Setup.1.0.4.exe`

## 3. Opublikuj

Kliknij **Publish release**.

## 4. Zaktualizuj stronę

Skopiuj zawartość `docs/wordpress/08-pobierz-1.0.4.html` lub `01-pobierz.html` i wklej jako blok Własny HTML na stronie produktu.
